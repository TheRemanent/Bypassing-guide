# Código de Conducta

Para mantener un ambiente respetuoso y constructivo:

- No se permite lenguaje ofensivo.
- No se tolera acoso o discriminación.
- Las discusiones deben mantenerse técnicas y respetuosas.
- Cualquier violación puede resultar en bloqueo del repositorio.

Este proyecto sigue el estándar de respeto y colaboración de la comunidad.
